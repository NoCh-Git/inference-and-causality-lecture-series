# inference-and-causality-lecture-series
---
This repository serves as the main entry point for all weekly materials of the course **Inference and Causality**.  
Each week has its own dedicated repository containing slides, notebooks, and requirements.
This course was originally designed for master's students in data science, big data and AI.

## Weekly Modules

| Week | Topic | Repository Link |
|------|--------|------------------|
| **1** | Statistical inference; Bayesian vs. frequentist; Bayes’ theorem; Bayesian networks | https://github.com/NoCh-Git/inference-and-causality-week1 |
| **2** | Continuous Bayes; Fisher information; conjugate priors; Monte Carlo & Markov chains; correlation vs. causation; Pearson correlation; Granger causality; DAG fundamentals | https://github.com/NoCh-Git/inference-and-causality-week2 |
| **3** | Observation vs intervention; confounders; proxy data; counterfactuals; RCTs | https://github.com/NoCh-Git/inference-and-causality-week3 |
| **4** | Frontdoor & backdoor adjustment (do-calculus) | https://github.com/NoCh-Git/inference-and-causality-week4 |
| **5** | Mediation & collider fallacies; Berkson’s paradox; Simpson’s paradox | https://github.com/NoCh-Git/inference-and-causality-week5 |
| **6** | Exam & open discussion session (no repository) | — |

---

## How to Use This Repository

1. Pick the week you need from the table above.  
2. Open the linked GitHub repository.  
3. Follow the instructions inside each weekly folder:
   - Run the Jupyter notebooks  
   - Review slides  
   - Complete and submit homework (where applicable)
